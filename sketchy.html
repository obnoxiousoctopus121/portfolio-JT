sketchy

at a glance

For my final project in CSCI:150: Introduction to Object Oriented Programming class, I designed a program that allows users to create digital art with Java. The program’s functionalities include a color customizable, cursor controlled pen tool, a resizable rectangle and ellipse generator, and undo/redo capabilities.

visual and design choices

Since this program relied heavily on a graphical user interface, it was important to make careful visual and program design choices. I made the following design choices that prioritized learnability, usability, and addressing user needs.

graphical user interface:

A major part of my program is the graphical user interface. I wanted to design a simple and straightforward interface so that it could have high learnability and usability. Utilizing JavaFX packages, the program interface consists of a blank white pane with a sidebar displaying drawing tools and functionalities available to the user.


interfaces:

My program was designed to include multiple functionalities to provide users with different options. Since there was more than one shape option, I created a Shapes Interface that I implemented in each shapes’ respective classes.

stack:

To allow users to undo and redo actions, I utilized a stack. This data structure stored the actions of users, so that commands could be pushed and popped to the discretion of the user. This data structure was unseen to users, but critical for visually representing the undo and redo functions.


key takeaways

Creating programs with multiple complex components requires careful design considerations. This project allowed me to practice both visual and program design thinking. When creating programs, it is important to keep users in mind. For this project, I placed emphasis on learnability and usability. It was critical to design and label the sidebar simply, so that users could immediately understand the functionality of each button. While visual design considerations are important for human centered design, program design considerations are equally as important. The program functionalities met users needs with the help of back-end design components, such as interfaces and stacks.
